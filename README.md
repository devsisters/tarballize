tarballize [![Docker Stats]][Docker Hub]
========
Make tarballs of given git repository including its submodules.

```bash
docker run "-v${PWD}:/a" devsisters/tarballize https://github.com/devsisters/prometheus-cpp v0.4.2-hotfix
```
